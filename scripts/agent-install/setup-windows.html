CFCC Puppet Setup

  1. Right-click this link and select "Save Target As".
  2. Open Explorer to the place that you saved (probably Downloads)
  3. Right-click the file and select "Run with PowerShell"
Puppet will now be automatically installed. You may get a prompt about setting an execution policy. Respond with "Yes".
You can run it by opening Command Prompt as Administrator and typing "puppet agent -t".
